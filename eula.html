Cobalt's User Agreement and Privacy Policy

Revision 1, date of publishing: September 16, 2024

  • Cobalt is distributed "as is".
    The application comes with ABSOLUTELY NO WARRANTY - developer is not responsible for any damages on your account or device.
    Cobalt is not endorsed, sponsored, approved, developed by Valve Corporation or related to it. All trademarks are registered and protected properties of respective owners.
  • Cobalt does not utilize own servers.
    To access data, Cobalt uses your Steam account credentials. User data is only stored on and transferred between the user's device and Steam servers, operated by Valve and covered by their own privacy policy and subscriber agreement. You must agree to this terms in order to use Cobalt.
  • Cobalt ensures that user authorization data is secure.
    • Every network request is encrypted. WebSocket connections are made on WSS protocol, making use of TLS encryption. HTTP connections are not allowed, requiring the usage of HTTPS.
    • Authorization data is encrypted. Access tokens and (optional, if set) Steam Guard options are saved in a storage encrypted by a unique generated key. Your username and password are never saved on the device and are only transmitted once per authorization attempt.
  • Cobalt may send diagnostic data by using third-party frameworks.
    For tracking application crashes, Firebase Crashlytics (a Google service) is used, that is covered by Crashlytics and App Distribution Terms.
    Crash reports do not include sensitive data, such as: user ID, user name or other unique variables that can identify the user.
    After signing in, user can opt-out of collecting such information by navigating to Profile -> Settings -> Privacy and unchecking "Send crash reports".
  • Cobalt never uses granted install-time or runtime permission outside specified functions.
    • "Camera" permission is used only for the QR code scanner feature in the Guard section. This permission is not required to use the application.
    • "Internet" permission is used to communicate with Valve or Firebase servers for the uses stated above. This permission is required for application functionality and is granted by the system automatically.
  • Cobalt is an open-source project.
    To verify the privacy claims, you can:
    • Visit Cobalt's GitHub page: https://github.com/itaysonlab/jetisteam
    • Visit kSteam's GitHub page: https://github.com/itaysonlab/ksteam
    kSteam is a separate project that is used in Cobalt to communicate with Steam servers.
  • If you have any other concerns, you can reach the developer.
    You can send an email to cobalt [at] vkx [dot] app (email address is obfuscated to prevent scraping). Your questions will be answered as soon as possible.
